## 首页短介绍

`Trading Agents Desktop` 是一个面向中文投资场景重构的 `TradingAgents` 桌面版 fork。它把上游的多智能体金融研究框架，改造成更适合实际使用的桌面应用：支持 `DeepSeek` 等模型，支持 A 股 6 位代码、中文股票名和板块词输入，并优先使用 `AkShare`、腾讯证券、东方财富、财新、百度财经等更适合中国网络环境的数据链路。

## 首页卖点

- 原生桌面端，不用只靠命令行
- 支持 A 股代码、中文股名、板块词
- 面向中国网络环境优化的数据链路
- 实时查看 Agent 状态、日志和最终报告
- 支持 DeepSeek / OpenAI / Claude / Gemini / Ollama
- 已补齐打包脚本和 GitHub Actions 自动构建
