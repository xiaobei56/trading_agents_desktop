# GitHub 截图清单

建议准备 4 张图，足够把这个仓库讲清楚。

## 1. 主界面总览

建议内容：

- 左侧完整配置区
- 右侧实时报告和最终报告
- 底部日志区

建议标题：

`桌面端主界面`

建议说明：

`在一个窗口里完成模型配置、A股分析和报告查看。`

## 2. A 股输入示例

建议内容：

- 输入 `600519`
- 输入 `中际旭创`
- 输入 `黄金板块`

建议标题：

`支持A股代码、中文股名和板块词`

建议说明：

`无需手动补交易所后缀，也支持中文查询。`

## 3. 实时执行状态

建议内容：

- Agent 状态表
- 活动日志
- 当前报告片段

建议标题：

`实时查看多智能体执行过程`

建议说明：

`能看到每个角色当前进度，而不是黑盒等待。`

## 4. 最终报告结果

建议内容：

- 完整报告页面
- A 股分析结论
- 结果导出路径

建议标题：

`分析完成后自动生成报告`

建议说明：

`最终报告会落盘到本地 results 目录，方便复盘和归档。`

## 截图风格建议

- 尽量选中文界面
- 用真实 A 股示例，不要用空白页
- 避免把 API Key 暴露在截图中
- 截图尺寸统一，优先横版
- 如果做 GIF，控制在 20 到 40 秒
