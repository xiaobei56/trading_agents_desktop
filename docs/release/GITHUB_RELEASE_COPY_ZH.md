# GitHub Release 文案模板

## Release Title

`v0.1.0 - Desktop A股版首发`

## Release Summary

这是 `Trading Agents Desktop` 的首个可发布版本，重点围绕中文/A 股场景做了大量适配。

本次版本包含：

- 原生桌面客户端
- 中文界面与更清晰的可读性优化
- `DeepSeek` 等多模型提供商支持
- A 股 6 位代码自动识别
- 中文股票名识别
- 中国市场板块/主题词查询支持
- 优先使用中国网络友好的数据链路
- 更友好的错误提示与 API Key 保存体验

## Highlights

### 1. 原生桌面端

不再需要频繁使用命令行，直接通过窗口配置股票、模型、研究深度和分析师组合，并实时查看日志与最终报告。

### 2. A 股适配

支持以下输入方式：

- `600519`
- `000001`
- `300308.SZ`
- `中际旭创`
- `黄金板块`

### 3. 中国链路优先

针对 A 股和中文查询，优先使用：

- AkShare
- 腾讯证券
- 新浪财经
- 东方财富新闻
- 财新头条
- 百度财经日历

### 4. 更适合中文模型

默认推荐 `DeepSeek`，同时保留 `OpenAI`、`Claude`、`Gemini`、`OpenRouter`、`Ollama` 等多提供商能力。
